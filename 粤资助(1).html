全学段学生资助政策体系（2026年现行）

📘 学前教育资助

资助对象：普惠性幼儿园家庭经济困难儿童、孤儿、残疾儿童

资助标准：1000元/生/年

评定时间：每年秋季学期9-10月集中评定，春季学期3-4月补报复核

评定条件：1.在经审批的公办/普惠性民办幼儿园就读；2.脱贫家庭、低保家庭、特困供养、孤儿、事实无人抚养、残疾儿童，或因病因灾因意外造成家庭经济困难儿童

评定流程：1.家长/监护人提交申请；2.幼儿园初审并校内公示；3.报属地教育主管部门审核备案；4.确定资助名单并发放资金

材料准备：1.学前教育资助申请表；2.幼儿及监护人户口本、身份证；3.家庭经济困难相关证明（低保证、特困证、脱贫证明、残疾证、孤儿证明等）；4.幼儿园就读证明

注意事项：需提供家庭困难相关证明材料，材料真实有效，公示无异议后方可认定

📘 义务教育阶段资助

资助内容：免学杂费、免费提供教科书，寄宿生生活补助，营养改善计划

资助标准：寄宿生补助（小学1000元/年，初中1250元/年）；营养改善计划4元/天

评定时间：秋季学期9-10月统一审核认定，春季学期3-4月动态调整、补发资助

评定条件：1.在校在籍义务教育阶段学生；2.脱贫家庭、低保、特困供养、孤儿、事实无人抚养、残疾学生，或家庭突发重大变故、重病、灾害等造成经济困难的学生

评定流程：1.学生/家长向学校提交申请；2.班级摸底、班主任初审；3.学校评审小组评审并校内公示；4.报属地教育局审核备案；5.落实免补政策、发放资助资金

材料准备：1.义务教育家庭经济困难学生资助申请表；2.户口本、监护人身份证；3.家庭经济困难证明（低保证、特困证、脱贫证明、残疾证、重大疾病诊断书、受灾证明等）；4.在校就读证明

注意事项：免学杂费、教科书为普惠性政策，生活补助需按困难等级评定，公示期不少于5个工作日

📘 普通高中资助

资助内容：国家助学金、免学费政策

资助标准：国家助学金1000、2000、3000元/年分档发放；免学费覆盖符合条件困难学生

评定时间：秋季9—10月集中评定，春季仅针对新增困难学生动态增补，不全面认定

评定条件：1.具有普通高中正式学籍的在校生；2.家庭经济困难（脱贫、低保、特困、残疾、孤儿等）；3.品行良好、遵守校纪校规、学习态度端正

评定流程：1.学生个人提交申请；2.班级评议、班主任初审；3.学校资助评审小组审核；4.校内公示无异议后报教育部门审批备案；5.发放资助资金、落实免学费

材料准备：1.普通高中国家助学金/免学费申请表；2.学生身份证、户口本；3.家庭经济困难相关证明材料；4.在校表现证明

📘 中职教育资助

资助内容：国家助学金、免学费、国家奖学金

资助标准：国家助学金2000元/年（一、二年级）；全日制在校生全部免学费；国家奖学金6000元/年

评定时间：秋季9-10月集中申请评定，春季3-4月针对新增困难学生动态增补

评定条件：1.全日制正式学籍中职在校生（助学金限一、二年级）；2.助学金：涉农专业学生或家庭经济困难学生；3.奖学金：品学兼优、综合表现突出；4.在校表现良好、遵守纪律、学籍有效

评定流程：1.学生向学校提交申请及材料；2.班级初审、学校资助部门审核；3.校内公示无异议后报教育/人社部门审批；4.确定资助名单并发放资金

材料准备：1.中职助学金/免学费/奖学金申请表；2.学生身份证、户口本；3.家庭经济困难证明材料（助学金）；4.学籍证明、在校成绩单及表现证明（奖学金）

📘 研究生资助

资助内容：国家奖学金、国家助学金、学业奖学金、三助一辅岗位补助、国家助学贷款等

资助标准：国家奖学金20000元/年；国家助学金不低于6000元/年；学业奖学金分档发放（院校自定）；三助一辅岗位补助按月发放

评定时间：每年9-12月集中评定各类奖学金、助学金，三助一辅岗位按学期认定，助学贷款暑假至开学初期办理

评定条件：1.全日制正式学籍研究生；2.品行端正、学业合格、无违纪记录；3.助学金：家庭经济困难学生；4.奖学金：科研能力突出、学业成绩优异；5.三助一辅：具备相应岗位工作能力

评定流程：1.学生个人提交申请及佐证材料；2.导师推荐、学院评审小组审核评议；3.学校资助管理中心审定并校内公示；4.报上级部门备案后发放资助资金/落实岗位

材料准备：1.研究生资助申请表（按类型填报）；2.身份证、学籍证明；3.家庭经济困难证明材料（助学金）；4.成绩单、科研成果、获奖证明（奖学金）；5.个人能力证明（三助一辅岗位）

注意事项：学业奖学金、三助一辅岗位评定标准由各高校自主制定，具体以就读院校通知为准

普通高中国家助学金（完整政策）

一、资助对象

具有正式学籍、家庭经济困难的普通高中在校生

二、评定条件

1. 热爱祖国，拥护中国共产党领导

2. 遵守宪法法律及校纪校规，无违纪记录

3. 诚实守信，道德品质优良

4. 勤奋学习，努力上进，学习态度端正

5. 家庭经济困难，生活简朴，符合广东省家庭经济困难认定标准

三、评定时间

秋季：9—10月集中申请认定，为全年主要批次

春季：仅针对新增家庭经济困难学生动态增补，不开展全面认定

四、评定流程

1. 学生向学校提交书面申请及相关证明材料

2. 班级组织民主评议，班主任完成初审并提出意见

3. 学校资助评审工作小组进行审核评议

4. 校内公示（不少于5个工作日），无异议后确定拟资助名单

5. 报属地教育行政部门审批、备案

6. 学校按审批结果发放助学金

五、材料准备

1. 普通高中国家助学金申请表（学校统一发放）

2. 学生本人身份证、户口本复印件（原件备查）

3. 家庭经济困难相关证明材料（低保证、特困人员救助供养证、脱贫家庭证明、残疾证、孤儿证明等）

4. 学生在校表现证明（班主任签字确认）

六、资助标准

分三档发放：1000元/年、2000元/年、3000元/年，根据家庭困难程度分档

七、注意事项

1. 需真实提供困难证明，不得弄虚作假、虚报瞒报

2. 公示期间有异议的，学校须在3个工作日内核查并反馈

3. 助学金资金直接发放至学生本人银行卡，专款专用

中职国家助学金（完整政策）

一、资助对象

全日制中职一、二年级涉农专业学生，或非涉农专业家庭经济困难学生

二、评定条件

1. 具有中等职业学校全日制正式学籍，在校正常就读

2. 遵守学校规章制度，品行良好，无违纪记录

3. 涉农专业学生直接认定，非涉农专业学生符合家庭经济困难认定标准

4. 积极参加学校各项教学活动，学业成绩合格

三、评定时间

秋季：9—10月集中申请、审核、认定

春季：3—4月针对新增家庭经济困难学生动态增补

四、评定流程

1. 学生向学校资助部门提交申请及相关材料

2. 班主任结合学生在校表现完成初审

3. 学校资助评审小组审核，确定拟资助名单

4. 校内公示（不少于5个工作日），接受师生监督

5. 公示无异议后，报教育/人社行政部门审批

6. 审批通过后，学校按月发放助学金

五、材料准备

1. 中等职业学校国家助学金申请表

2. 学生本人身份证、户口本复印件（原件备查）

3. 家庭经济困难证明材料（低保证、特困证、脱贫证明等，非涉农专业需提供）

4. 学生学籍证明（学校学籍管理部门出具）

六、资助标准

2000元/生/年，按10个月计发，200元/生/月；全日制中职在校生全部享受免学费政策

七、注意事项

1. 学籍须在教育部全国中等职业学校学生管理信息系统正常注册

2. 休学、退学、转学学生暂停发放助学金，恢复学籍后重新审核

3. 严禁代领、冒领助学金，一经发现取消资助资格并追回资金

本专科生国家助学金（最全官方版）

一、资助对象

全日制普通本专科（高职、专科）在校生中家庭经济困难学生

二、评定条件

1. 热爱祖国，拥护中国共产党的领导，树立正确的世界观、人生观、价值观

2. 遵守宪法和法律，遵守学校规章制度，无任何违纪违法记录

3. 勤奋学习，积极上进，学业成绩合格，无不及格科目

4. 家庭经济困难，生活简朴，已完成学校家庭经济困难学生认定

5. 诚实守信，品行端正，积极参与学校公益活动者优先

三、评定时间

秋季认定：9—10月（全学年主要认定批次，覆盖全年资助）

春季增补：3—4月（漏评、新增困难、家庭变故、困难程度加重学生均可申请）

四、评定流程

1. 学校发布资助申请通知，学生通过校内资助系统线上提交申请，同时提交纸质材料

2. 班级成立评议小组，开展民主评议，提出初步评议意见

3. 学院资助工作小组对班级评议结果进行初审，确定学院拟资助名单

4. 学院公示（不少于3个工作日），无异议后报学校学生资助管理中心

5. 学校资助管理中心复核，全校公示（不少于5个工作日）

6. 公示无异议后，报省级教育行政部门备案，学校发放资助资金

五、材料准备

1. 广东省家庭经济困难学生认定申请表（原件）

2. 学生本人身份证、户口本复印件（原件备查）

3. 家庭经济困难相关证明材料（低保证、特困人员救助供养证、脱贫家庭监测证明、残疾证、重大疾病诊断书等）

4. 学生本人银行卡复印件（开户行、卡号、姓名清晰，与身份证一致）

六、资助标准

平均3700元/生/年，各高校分2-3档发放，具体档级金额由高校结合实际制定

七、注意事项

1. 严禁弄虚作假、虚报困难情况，一经查实取消资助资格，记入学生诚信档案

2. 资助资金直接发放至学生本人银行卡，仅限用于学习和生活开支，不得挪作他用

3. 在校期间受到纪律处分的学生，取消当学年助学金评定资格

4. 助学金按学期发放，每学期发放一次

本专科生国家助学贷款（最全官方版）

一、资助对象

家庭经济困难的全日制普通本专科生、研究生、第二学士学位学生

二、评定条件

1. 具有中华人民共和国国籍，且持有中华人民共和国居民身份证

2. 诚实守信，遵纪守法，无不良信用记录，无违法违纪行为

3. 家庭经济困难，不足以支付在校期间完成学业所需的学费、住宿费

4. 入学前户籍在广东省内（生源地助学贷款），或就读于广东省内高校（校园地助学贷款）

5. 全日制在校学生，学籍状态正常

三、评定时间

首次贷款/续贷：每年7月1日—9月30日（暑假期间），开学初期可补办

校园地助学贷款：按就读高校通知时间办理

四、评定流程（生源地信用助学贷款）

首次办理：1.学生登录国家开发银行助学贷款信息网注册申请；2.学生与共同借款人携带材料到户籍所在地教育局学生资助管理中心现场审核、签订合同；3.高校确认学籍后，贷款资金发放至高校学费账户，剩余资金转至学生个人账户

续贷办理：1.学生线上提交续贷申请；2.无需现场办理，线上审核通过后签订电子合同；3.贷款资金直接发放至高校及学生账户

五、材料准备

首次办理：1.学生本人身份证、户口本原件及复印件；2.共同借款人身份证、户口本原件及复印件；3.录取通知书/学生证原件及复印件；4.国家开发银行助学贷款申请表（网上打印）

续贷办理：1.学生本人身份证原件；2.续贷申请表（网上打印）；3.共同借款人身份证复印件（如信息无变更）

六、贷款额度与用途（2024年秋季学期起执行新标准）

贷款额度：本专科生每人每年最高20000元，最低1000元，研究生每人每年最高25000元，最低1000元

贷款用途：优先用于支付学费和住宿费，超出部分可用于弥补日常生活费。

七、利息政策

在校期间：国家全额贴息，学生不支付任何利息

毕业当年：9月1日起开始自行承担利息，按年计息

还款宽限期：毕业后5年内为还本宽限期，仅需支付利息，5年后开始偿还本金+利息

八、注意事项

1. 贷款必须用于学业相关开支，不得挪作他用，一经发现停止贷款发放

2. 毕业前必须登录助学贷款系统完成毕业确认，更新个人联系方式、就业信息

3. 按期足额还款，维护个人征信记录，逾期还款将产生罚息并记入个人信用档案

4. 升学、转学、休学、退学的学生，须及时到教育局及高校办理贷款变更手续
